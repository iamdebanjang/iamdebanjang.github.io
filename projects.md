<h2 style="margin: 60px 0px -15px;">Projects</h2>

### MirageNet: Physics-aware Mirage Segmentation
Developed a physics-aware deep learning framework for detecting mirage-like optical distortions in natural scenes using depth estimation, flow modeling, and segmentation.

### MediVLM: Vision-Language Model for Radiology
Built a multimodal architecture combining Faster R-CNN, CLIP encoders, and a GPT-based decoder for radiology report generation from chest X-ray images.

### Active Learning for Image Segmentation
Designed an active learning pipeline using binary user feedback to reduce annotation effort while improving segmentation performance.

### Graph Neural Network Watermarking
Proposed a watermarking framework for GNNs using Node2Vec embeddings and subgraph perturbation strategies for model ownership verification.

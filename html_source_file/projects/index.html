Projects

MirageNet: Physics-aware Mirage Segmentation

Developed a physics-aware deep learning framework for detecting mirage-like optical distortions in natural scenes. The model integrates depth estimation with flow-based modeling to capture refraction effects and improve segmentation accuracy.

MediVLM: Vision-Language Model for Radiology

Built a multimodal architecture combining Faster R-CNN for region extraction, CLIP encoders for visual-text alignment, and a GPT-based decoder for automated radiology report generation from chest X-ray images.

Active Learning for Image Segmentation

Designed an efficient active learning pipeline using binary user feedback to minimize labeling effort while improving segmentation performance across challenging datasets.

Graph Neural Network Watermarking

Proposed a watermarking mechanism for Graph Neural Networks using Node2Vec embeddings and subgraph perturbation strategies to ensure model ownership verification and robustness.
